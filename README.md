# Upload Big Files using AWS

Esse repositório foi parte de estudos de AWS onde o objetivo era subir um arquivo grande no formato `.pdf` na AWS e depois poder resgatar esse arquivo.

## Ferramentas Utilizadas
- AWS CDK
- AWS CloudFront
- AWS Gateway API
- AWS Lambda
- AWS S3
